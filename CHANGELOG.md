# История изменений

## 0.7.1 — 2026-07-17

- упрощена вкладка «Диктовка»: повторяющийся заголовок удалён, запись и
  последний результат объединены в одну рабочую карточку, а лишние действия
  «Повторить», «Отменить вставку» и преобразования убраны с главного экрана;
- раздел распознавания теперь действительно переключается между режимами
  «Авто», «Онлайн» и «Локально»; в локальном режиме можно вручную выбрать модель
  и скорость, а в остальных режимах безопасный профиль подбирается компьютером;
- выбранный локальный профиль больше не перезаписывается при следующем запуске,
  и финальная расшифровка использует именно выбранную модель;
- кнопки микрофона получили однозначные названия и видимый результат обновления
  списка; техническая диагностика теперь явно обозначена как копирование;
- обычные параметры оформления отделены от медленной обработки через Ollama,
  которая спрятана в раскрываемый дополнительный блок с объяснением;
- «Сниппеты» переименованы в понятные «быстрые фразы», добавлены описание и
  готовый пример; профили программ скрыты в блок для опытных пользователей;
- включение истории перенесено на вкладку «История» и применяется сразу.
- номер, размер и ссылка на установщик на лендинге теперь автоматически
  загружаются из последнего GitHub Release и больше не обновляются вручную;
- после слияния новой версии в `main` GitHub автоматически создаёт тег, собирает
  и публикует единственный установщик `Rechka-Setup-X.Y.Z.exe`;
- старое пользовательское имя VoiceInput удалено из сборки, автозапуска и
  каталога данных; существующие настройки автоматически переносятся в
  `%LOCALAPPDATA%\Rechka`.

## 0.7.0 — 2026-07-17

- добавлен автоматический анализ процессора, памяти и доступного ускорителя:
  слабые компьютеры получают Tiny, остальные — встроенную Base с подходящим
  числом потоков;
- на главной странице появилась постоянная карточка с активным способом
  распознавания, локальным резервом, характеристиками компьютера и измеренной
  скоростью последней расшифровки;
- добавлено бесплатное распознавание через публичные Hugging Face Spaces без
  регистрации и API-ключа; на слабых ПК оно используется первым, а на более
  быстрых становится резервом для локального Base;
- локальная модель загружается лениво и больше не занимает память, пока работает
  сетевой режим;
- Tiny и Base включаются в установщик, поэтому локальный резерв не зависит от
  первой загрузки модели;
- фоновая многократная расшифровка черновика отключена: она конкурировала с
  финальной обработкой за одну модель и могла создавать впечатление зависания
  на слабых CPU;
- локальная обработка оставляет одно физическое ядро Windows и интерфейсу;
- сетевой запрос имеет жёсткий дедлайн 60 секунд даже при heartbeat-сообщениях
  сервера, после чего включается локальный резерв;
- новые публичные версии стали обязательными: уведомление выводится поверх
  скрытого окна и предлагает только обновить программу или закрыть её;
- проверка обновлений повторяется каждые шесть часов, а после сетевой ошибки —
  через десять минут; GitHub Actions явно помечает выпуск как Latest и проверяет
  доступность установщика, размера и SHA-256 через канал обновлений;
- добавлена проверка готовности системного распознавания Windows. Текущий
  классический установщик не даёт необходимую пакетную идентификацию, поэтому
  этот путь включится только после перехода на совместимую упаковку;
- политика конфиденциальности теперь явно описывает отправку аудио выбранному
  публичному серверу и автоматический локальный резерв.

## 0.6.2 — 2026-07-17

- из компактной плашки записи убран постоянный пустой блок предпросмотра:
  живой черновик появляется только после распознавания надёжной фразы, занимает
  не больше двух строк и использует основную модель распознавания (по умолчанию
  Base), как и финальный текст;
- установщик больше не растягивает системные кнопки вручную, поэтому рамка
  Windows рисуется целиком и одинаково со всех сторон;
- основной установщик переименован в `Rechka-Setup-X.Y.Z.exe`; прежнее имя
  публикуется только как совместимая копия для обновления старых версий;
- обновлятор предпочитает новое имя установщика, но принимает прежнее для
  бесшовного перехода.

## 0.6.1 — 2026-07-17

- неточный черновик отдельной Tiny-модели удалён; финальное распознавание
  начинается после остановки записи;
- финальная обработка режима «Общение» аккуратно сохраняет очевидные вопросы,
  приводит обычный дефис между частями фразы к русскому тире и завершает
  непунктуированные фразы;
- добавлены голосовые команды «поставь тире» и «поставь дефис», исправлена
  команда «поставь точку с запятой»;
- основная горячая клавиша сокращена до `Ctrl + Пробел`; прежний стандарт
  `Ctrl + Alt + Пробел` автоматически обновляется, а пользовательские сочетания
  сохраняются;
- рекомендуемый профиль распознавания изменён на `Base + Баланс`: небольшая
  дополнительная задержка даёт аккуратнее текст, а режим «Быстро» остаётся
  доступен вручную;
- Tiny-модель удалена из установщика: приложение быстрее подготавливается к
  работе и занимает меньше места;
- `Rechka.exe` теперь содержит версию и название продукта в свойствах Windows;
- проверка обновлений поддерживает безопасные перенаправления GitHub после
  переименования репозитория.

## 0.6.0 — 2026-07-17

- добавлен VAD до Whisper: тишина и устойчивый фоновый шум не создают
  выдуманную расшифровку;
- запись стала устойчивее к тихому сигналу, клиппингу, смене микрофона и длинным
  диктовкам; для предпросмотра в памяти остаются только последние 20 секунд;
- добавлена трёхсекундная проверка микрофона и локальная диагностика без аудио,
  текста и словаря пользователя;
- приложение автоматически выбирает оптимальный CPU-профиль или CUDA и безопасно
  возвращается на CPU, если ускоритель недоступен;
- `Esc` отменяет текущую запись без расшифровки и вставки;
- добавлены повтор записи, отмена последней вставки и голосовые команды
  «отмени», «повтори» и «нажми Enter»;
- после вставки восстанавливается прежнее содержимое буфера обмена со всеми MIME-
  форматами;
- результат разделён на готовый текст и исходную расшифровку; появился режим
  «Дословно» и пользовательский локальный режим обработки;
- добавлены точные голосовые сниппеты и профили приложений с отдельными режимами
  и терминами;
- добавлена отключённая по умолчанию локальная история без аудио, а также её
  просмотр, копирование и полное удаление;
- добавлены импорт и экспорт персонализации, не включающие историю;
- добавлены быстрые преобразования результата в сообщение, письмо, список и
  чек-лист задач;
- главное окно получило цельную полноширинную навигацию, отдельные состояния
  записи с таймером и уровнем голоса, а настройки — стабильную сетку и
  закреплённую панель сохранения;
- поле результата автоматически растёт вместе с текстом до безопасной высоты,
  после чего включает собственную прокрутку; готовый текст можно поправить, а
  копирование работает с открытой вкладкой результата;
- прокрутка страницы больше не переключает микрофон и другие закрытые списки;
- первый запуск получил короткий онбординг с выбором и проверкой микрофона;
- добавлены benchmark-инструменты с WER, CER, RTF, разрезами по тегам и отдельной
  проверкой галлюцинаций на тишине и шуме.

## 0.3.4 — 2026-07-16

- длинные записи больше не обрываются после первых 30 секунд;
- живой черновик обновляется частями и автоматически прокручивается к новым словам;
- исправлено открытие настроек из уже запущенного экземпляра приложения;
- полностью обновлён интерфейс главного окна, настроек и плашки записи;
- добавлен явный прогресс загрузки обновления внутри приложения;
- обновление закрывает текущую версию, тихо заменяет файлы и запускает новую;
- установщик получил фирменное оформление «Речки» и не требует вручную закрывать приложение.

## 0.3.3 — 2026-07-16

- стандартные и пользовательские глобальные горячие клавиши;
- безопасная смена комбинации без потери предыдущей рабочей клавиши;
- более чувствительная дорожка микрофона, таймер и явный статус записи голоса;
- быстрый инкрементальный черновик на отдельном экземпляре Whisper;
- финальная расшифровка больше не ждёт занятую обработку предпросмотра;
- старые тяжёлые настройки Turbo + Ollama переводятся на рекомендуемый быстрый профиль;
- исполняемый файл переименован в `Rechka.exe`, чтобы Windows создала новый ярлык без старого кэша иконки.

## 0.3.2 — 2026-07-16

- новое название «Речка» и цельная визуальная айдентика;
