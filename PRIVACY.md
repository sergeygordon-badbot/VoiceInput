# Конфиденциальность «Речки»

«Речка» предлагает три способа распознавания: «Авто», «Онлайн» и «Локально».
В автоматическом режиме программа проверяет бесплатный публичный Hugging Face
Space и подбирает основной способ по характеристикам компьютера. В режиме
«Онлайн» доступный Space используется первым, а локальная модель остаётся
резервом. В режиме «Локально» сетевые службы распознавания не проверяются и
аудио не покидает компьютер. Когда выбран сетевой способ, записанная фраза
отправляется непосредственно публичному сервису и возвращается в виде текста.
Разработчик «Речки» не получает аудио, не проксирует его через свой сервер и не
использует общий API-ключ.

Публичный Space принадлежит третьей стороне. Его доступность, правила хранения,
очереди и лимиты определяются владельцем Space и платформой Hugging Face.
Использовать такой режим для конфиденциальной речи не рекомендуется.

Если публичный сервер недоступен, не отвечает вовремя или отклоняет запрос,
программа автоматически использует локальный Whisper. В локальном режиме аудио
не покидает компьютер. Tiny и Base входят в установщик, поэтому этот резерв
работает без регистрации и оплаты.

Программа обращается в интернет только в следующих случаях:

- проверка публичных Hugging Face Spaces в режимах «Авто» и «Онлайн» и отправка
  записанной фразы выбранному Space, когда сетевой способ стал активным;
- скачивание дополнительной локальной модели с Hugging Face, если пользовательская
  сборка не содержит подходящих файлов;
- проверка и скачивание обновлений из публичного GitHub Releases;
- работа с локальной Ollama по адресу `127.0.0.1`, если пользователь включил эту
  функцию. Ollama не является облачным сервисом «Речки».

«Речка» не содержит аналитики, рекламных идентификаторов и телеметрии. Настройки
хранятся в `%LOCALAPPDATA%\Rechka`. Распознанный текст может кратковременно
попадать в буфер обмена Windows при вставке через `Ctrl+V`; после вставки программа
восстанавливает предыдущее содержимое буфера.

Локальная история распознаваний выключена по умолчанию. После явного включения она
хранит в `%LOCALAPPDATA%\Rechka\history.json` не более 100 последних исходных и
обработанных текстов, выбранный режим и имя приложения. Аудио в историю не
записывается. Историю можно полностью удалить из интерфейса. Экспорт
персонализации не включает историю.

Техническая диагностика не содержит аудио, расшифровку или пользовательский
словарь. Она может содержать выбранный тип распознавания, сведения о процессоре,
объёме памяти, версии Windows и доступности ускорителя.

Редакция политики: 17 июля 2026 года. Она меняется только при изменении самой
политики, а не при каждом выпуске приложения.
