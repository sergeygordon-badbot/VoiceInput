# Безопасность

Поддерживаемая ветка публичной beta: последняя опубликованная версия «Речки».

Для сообщения об уязвимости используйте приватное сообщение об уязвимости
GitHub Security Advisory в репозитории проекта. Не публикуйте ключи, персональные
данные, аудиозаписи и другие чувствительные материалы в обычных Issues.

Обновления «Речки» принимаются только из настроенного публичного GitHub
репозитория. Перед запуском установщика программа проверяет имя файла, HTTPS-домен,
размер и SHA-256, предоставленный GitHub Releases.
